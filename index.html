Square

A simple, clean watchface for the Pebble smartwatch

Square is a feature-packed watchface with a clean design. Since 2015, Square has constantly been updated with new user-requested features such as new languages and fitness tracking. A full list of features can be found on the Pebble app store linked below. It is a native pebble application, written in C and using JavaScript for settings and weather API functions.

Square is featured on the front page of the Pebble app store, and currently has over 50,000 installs and 1,600 hearts. Users from all over the world have been in touch to say how much they enjoy it, and some have even donated money to support further development.

greenlist

Track your food waste and make smarter purchasing decisions

greenl.ist is a simple grocery list webapp that allows users to track how they use their groceries and see efficiency data displayed right on their shopping list. Using this info, users can adjust their purchasing habits to reduce their food waste and save money. Users can also create shared grocery lists with their own efficiency datasets to track their habits as a household or group.

greenl.ist was created as a school project at the British Columbia Institute of Technology in an effort to help with Metro Vancouver's Zero Food Waste Program. Students were tasked with developing web applications to engage users with the zero waste effort, and make helping the environment more accessible and interesting. It was created using AngularJS, Google Firebase, and AngularFire.

NextBus Vancouver

Live bus arrival info on your Pebble smartwatch

NextBus Vancouver is a Pebble watchapp that takes advantage of Metro Vancouver's TransLink NextBus system to let users quickly get information about the next arrival at a bus stop. By entering the stop ID listed on any TransLink bus stop sign, users can see what the next bus to arrive is and how many minutes it will be until it arrives.

NextBus Vancouver uses the TransLink API and takes advantage of real-time GPS data for TransLink vehicles to give accurate arrival estimations. It is a native Pebble app written in C and using JavaScript for API interfacing.
