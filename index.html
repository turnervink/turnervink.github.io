A few of my projects:

Square

A feature-filled watchface for the Pebble smartwatch. I started Square as a simple, no-frills watchface, and it quickly grew into something much bigger. By far my most advanced watchface for Pebble, Square has helped me learn C and JavaScript as I added features like multilingual support and weather information. I also got to learn all sorts of aspects of the Pebble SDK, including integrating step tracking and battery status. Download it for your watch from the Pebble app store or check out the source on GitHub.

NextBus Vancouver

NextBus Vancouver is a simple watchapp I created to learn the API for Metro Vancouver's public transit authority, TransLink. A user can input the stop number of any TransLink bus stop and receive information on the next arrival. Currently NextBus Vancouver is very simple, and I hope to add more features to it in the future. Download it on the Pebble app store.
