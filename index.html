My projects

Square

Square is a simple, clean watchface for the Pebble smartwatch that offers a wide variety of customization options. Users can choose colours, language, date/time format, and can choose to display a weather forecast. The middle bar can function as a static accent, a battery meter, or show progress towards a daily step goal. Square is written in C for the native application, and uses JavaScript for fetching weather information and user configuration. It utilizes Pebble Clay for offline configuration. You can download Square for your Pebble on the Pebble App Store and view the source code on GitHub.

greenl.ist

greenl.ist is a simple grocery list webapp that allows users to track how they use their groceries and see efficiency data displayed right on their shopping list. Using this info, users can adjust their purchasing habits to reduce their food waste and save money. Users can also create shared grocery lists with their own efficiency data to track their habits as a household. greenl.ist was created as a school project at the British Columbia Institute of Technology in an effort to help with Metro Vancouver's Zero Food Waste Program. It was created using AngularJS, Google Firebase, and AngularFire. Information on the team behind greenl.ist can be found on GitHub, and you can try greenl.ist here.

NextBus Vancouver

NextBus Vancouver is a Pebble watchapp that takes advantage of Metro Vancouver's TransLink NextBus system to let users quickly get information about the next arrival at a bus stop. By entering the stop ID listed on any TransLink bus stop sign, users can see what the next bus to arrive is and how many minutes it will be until it arrives. NextBus Vancouver uses the TransLink API and takes advantage of real-time GPS data for TransLink vehicles to give accurate arrival estimations. You can get NextBus Vancouver on the Pebble App Store and view the source code on GitHub.
